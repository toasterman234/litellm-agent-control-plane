Every action available in the UI is also available over the REST API. Authenticate with `Authorization: Bearer <key>`.

## Base URL

| Environment | URL |
|---|---|
| Production | `https://litellm-rust.onrender.com` |
| Local | `http://localhost:4000` |

## Create an agent

```bash
curl -X POST $LAP_URL/api/v1/managed_agents/agents \
  -H "Authorization: Bearer $MASTER_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-agent",
    "runtime": "cursor",
    "model": "claude-opus-4-5",
    "system_prompt": "You are a helpful assistant."
  }'
```

## Start a session

```bash
SESSION=$(curl -s -X POST $LAP_URL/api/v1/sessions \
  -H "Authorization: Bearer $MASTER_KEY" \
  -H "Content-Type: application/json" \
  -d '{"agent_id": "<agent-id>"}' | jq -r .id)
```

## Send a message

```bash
curl -X POST $LAP_URL/api/v1/sessions/$SESSION/events \
  -H "Authorization: Bearer $MASTER_KEY" \
  -H "Content-Type: application/json" \
  -d '{"type": "human_turn", "content": "Summarize the README."}'
```

## Stream the response

```bash
curl -N "$LAP_URL/api/v1/sessions/$SESSION/events/stream" \
  -H "Authorization: Bearer $MASTER_KEY"
```

Events are server-sent (SSE), newline-delimited:

```
data: {"type":"session.status_running"}
data: {"type":"agent.message","content":"Here is the summary..."}
data: {"type":"session.status_idle"}
```

## SDK examples

<CodeGroup>

```python Python
import requests

session = requests.post(
    f"{LAP_URL}/api/v1/sessions",
    headers={"Authorization": f"Bearer {MASTER_KEY}"},
    json={"agent_id": agent_id},
).json()

requests.post(
    f"{LAP_URL}/api/v1/sessions/{session['id']}/events",
    headers={"Authorization": f"Bearer {MASTER_KEY}"},
    json={"type": "human_turn", "content": "Hello!"},
)
```

```javascript JavaScript
const session = await fetch(`${LAP_URL}/api/v1/sessions`, {
  method: "POST",
  headers: { Authorization: `Bearer ${MASTER_KEY}`, "Content-Type": "application/json" },
  body: JSON.stringify({ agent_id: agentId }),
}).then(r => r.json());

const stream = await fetch(`${LAP_URL}/api/v1/sessions/${session.id}/events/stream`, {
  headers: { Authorization: `Bearer ${MASTER_KEY}` },
});
```
